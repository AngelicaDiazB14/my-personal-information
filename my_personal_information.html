Simplex

Description: This project implements the Simplex algorithm to solve linear programming problems, allowing users to input an objective function and constraints. It handles multiple solutions, infeasibility, and unbounded problems.

Created by: Angélica Díaz and David Centeno

Tokenizer and Parser

Description: This project involves implementing the early stages of a compiler for the Triangle language in Rust. It includes a tokenizer and syntactic analysis, with a syntax tree visualization.

Created by: Angélica Díaz, Katerine Guzmán, Rafael Vargas, Joseph David Santamaría, and David Brenes

Test Plan

Description: This project involves developing a test plan for a web application managing activities for first-year students at Tecnológico de Costa Rica, focused on static and dynamic testing.

Created by: Angélica Díaz, David Centeno, Kevin Carranza, Aivy Masis

Transportation Algorithms

Description: This project applies transportation algorithms to solve transportation problems, including methods like Northwest Corner and MODI, and handles degenerate and unbalanced problems.

Created by: Angélica Díaz and David Centeno

Mastermind

Description: This project implements the classic game of Mastermind in Prolog, where players can either guess or generate a sequence of colors with up to 10 attempts.

Created by: Angélica Díaz

Queue Management System

Description: A system designed to manage queues in physical service locations. It prioritizes special customers and provides statistics on waiting times and service areas.

Created by: Angélica Díaz, David Centeno, Dina Monge

Pyramid Solitaire

Description: This project implements a Pyramid Solitaire game in Rust, where players must clear the pyramid using basic card removal rules.
